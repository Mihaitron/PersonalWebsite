About Me

I'm an extremely passionate person, ready to take on any challenge at any time. I've started my game developer journey in high school, in a game dev club of sorts.

Now, nearly a decade later, I work as a game developer, bringing the same joy I feel whenever I play to other people around the globe.

Some of my personal technology-related passions include VR and 3D printing, but I'm also an extremely creative individual, dabbling in origami making as well as painting miniatures.

Yours Truly

Projects

Who Axed You?

Created for RGDA's Tales of Us game jam, this is a game about Vasile, a drunk peasant from an unnamed village fighting the legendary Romanian myths of weather wizards riding dragons. This game features 2 types of enemies: the weather wizard and the dragon, as well as a boss. It also features items, an inventory and cool axe-throwing mechanics!

Pain is a Lie

A submission for Brackey's Game Jam 2022.1, this puzzle game is a Sky, a woman who lost her loved one and feels the need to move. The game contains Sky's full story as well as box-driven puzzle mechanics in a minimalistic, low-poly style.

Climbing Simulator 2022

As a submission to the Pacifist Jam, this is a game about my newest passion: climbing. In this game you have to climb as high as possible by pressing the keys assigned to the holds in your proximity while managing your stamina and chalk levels.

Death Doesn't Spare Cults

This game was made for the Wowie Jam 3.0. It features a full 3D environment, voice lines as well as multiple enemies and a boss. It also has 3 different endings, the secret one being quite tricky to get.

The Adventures of Edgar

Being a submission for Brackey's Game Jam 2020.1, you'd never guess this was my first ever game made for a jam. This is a quick puzzler in which the player has to guide Edgar the ferret back home through the tunnels he dug. It features multiple levels, the last one being the most challenging, set in a city sewer.

Here I achieved my dream of being a proper game developer, I learnt about the video games industry, working directly with the client as well as the difficulties that come with this. This is also where I learnt better code architecture for games.
Tractor, Set, GO! is my current workplace!
